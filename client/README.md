
## ✏️ How to use

  ### 1. Cloning
  ```
  $ https://github.com/SO-TY/bc-frontend.git
  ```

  ### 2. Install Packages & Run ⚛️
  - install packages
    ```
    $ cd bc-frontend
    $ npm install
    ```
  - run develop cilent
    ```
    $ npm start
    ```
    Checkout web service at http://localhost:3000
